# Quantitative-Analysis-NVIDIA-NVDA-vs.-Bitcoin
This project provides a comparative quantitative analysis of *NVIDIA* and *Bitcoin* during the 2025-2026 period. The goal is to challenge market assumptions regarding volatility and investment efficiency by calculating risk-adjusted returns using the *Sharpe Ratio*.

## Key Insights
* *The Volatility Twist:* NVIDIA's annualized volatility (*50.50%) has officially surpassed Bitcoin's (47.68%*), driven by the AI sector's rapid growth and market noise.
* *Efficiency Leader:* Despite higher volatility, NVIDIA proves to be the more efficient asset with a Sharpe Ratio of *1.52, compared to **0.79* for Bitcoin.

## Methodology
To ensure institutional-grade accuracy, a dual-calendar approach was implemented:
* *Equities ($NVDA):* Annualized based on a 252-day trading calendar.
* *Cryptocurrencies ($BTC):* Annualized based on a 365-day trading calendar.
* *Risk-Free Rate:* Adjusted to a 4% benchmark for calculation consistency.

## Tech Stack
* *Python 3.x*
* *yFinance API:* For real-time and historical financial data extraction.
* *Pandas & NumPy:* For data manipulation and vectorized mathematical operations.
* *Matplotlib/Seaborn:* For financial data visualization.
